**FluxCap** is an independent desktop client designed to connect seamlessly with the Fluxer network. Built with Electron, it provides a dedicated, fast, and customizable user interface for Windows, taking Fluxer out of your browser and directly onto your desktop.

## ğŸ›  Installation & Setup

### Standard Installation (Recommended)

1. [cite_start]Go to the [Releases](https://github.com/ByAldon/FluxCap/releases) page. 
2. [cite_start]Download the latest `FluxCap-Setup-1.4.8.exe` installer. 
3. [cite_start]Run the installer and follow the prompts. 

### Build from Source

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/ByAldon/FluxCap.git](https://github.com/ByAldon/FluxCap.git)
    ```
2.  **Install dependencies & Start:**
    ```bash
    npm install
    npm start
    ```

## ğŸ§¼ Clean Installation Guide

[cite_start]We highly recommend a clean installation when moving to FluxCap 1.4.8 to avoid conflicts with older "Unofficial Fluxer Client" files. [cite: 2, 3]

1.  [cite_start]**Uninstall:** Use **Revo Uninstaller** to remove the previous version. [cite: 2, 3]
2.  [cite_start]**Scan:** Perform a "Moderate" scan to delete leftover files in `%AppData%\unofficial-fluxer-client`. [cite: 2, 3]
3.  [cite_start]**Install:** Run the new FluxCap setup. [cite: 2, 3]
