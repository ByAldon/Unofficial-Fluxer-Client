# Unofficial Fluxer Client

An unofficial, open-source desktop client for Fluxer, built with Electron.

## Features
* **Interactive Installer**: NSIS-based setup wizard.
* **License Agreement**: Mandatory checkbox acceptance.
* **Tray Support**: Minimize to tray with cache clearing options.

## Development
1. Clone the repo
2. `npm install`
3. `npm start` (to test)
4. `npm run build` (to create installer)

## Disclaimer
This is an unofficial client and not affiliated with Fluxer.
