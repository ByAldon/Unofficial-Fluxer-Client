## üöÄ About the Project

**FluxCap** is an independent desktop client designed to connect seamlessly with the Fluxer network [cite: 2026-02-22]. Built with Electron, it provides a dedicated, fast, and customizable user interface for Windows, taking Fluxer out of your browser and directly onto your desktop [cite: 2026-02-22].

* **üîå Seamless Connectivity:** Connects directly to the official `web.fluxer.app` interface [cite: 2026-02-22].

* **üîÑ Auto-Updates:** Built-in automatic updates ensure you are always running the latest version [cite: 2026-02-22].
* **üíª Native Desktop Experience:** Features a system tray icon, custom window framing, and secure external link handling [cite: 2026-02-22].

## üõ† Installation & Setup

### Standard Installation (Recommended)

1. Go to the [Releases](https://github.com/ByAldon/FluxCap/releases) page [cite: 2026-02-22].
2. Download the latest `FluxCap-Setup-1.4.9.exe` installer [cite: 2026-02-22].
3. Run the installer and follow the prompts [cite: 2026-02-22].

### Build from Source

To get a development environment running or to build the application yourself, follow these steps [cite: 2026-02-22]:

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/ByAldon/FluxCap.git](https://github.com/ByAldon/FluxCap.git)
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd FluxCap
    ```
3.  **Install the dependencies:**
    ```bash
    npm install
    ```
4.  **Start the application in development mode:**
    ```bash
    npm start
    ```
5.  **Build the Windows Installer (.exe):**
    ```bash
    npm run build
    ```

### Clean Installation Guide

When updating to FluxCap 1.4.9, we highly recommend performing a clean installation to prevent old cache files or outdated configurations from causing unexpected behavior [cite: 2026-02-22].

1.  **Uninstall:** Use a tool like **Revo Uninstaller** to remove the previous "Unofficial Fluxer Client" [cite: 2026-02-22].
2.  **Scan for Leftovers:** Ensure remaining files in `%AppData%\unofficial-fluxer-client` are removed [cite: 2026-02-22].
3.  **Install the New Update:** Download and install the latest FluxCap release [cite: 2026-02-22].

## üíª Usage

Simply launch **FluxCap** from your start menu or desktop shortcut [cite: 2026-02-22]. The client will automatically connect to the Fluxer web interface [cite: 2026-02-22].

If you close the window, FluxCap will silently minimize to your system tray to keep you connected in the background [cite: 2026-02-22]. Right-click the tray icon to completely quit the application, clear the cache, or manually check for updates [cite: 2026-02-22].
